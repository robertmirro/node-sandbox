Browserify - math.js / math_browserify.js

Build math_browserify.js as follows:

$ browserify math.js -r ./square.js:mathSquare -o math_browserify.js

  • Include math.js, which will be invoked upon loading math_browserify.js
  • -require square.js so that math_browserify.html can require('mathSquare') and invoke its logic
