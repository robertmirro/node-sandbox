# Webpack Article
=======

  This is example code for [Webpack Usage](http://blog.madewithlove.be/post/webpack-your-bags/).

## Usage
  * Step1: Install dependencies
```shell
  npm install
```

  * Step2: Run server
```shell
  webpack-dev-server --inline --hot
```

  * Step3: Visit url
```shell
  http://localhost:8080/webpack-dev-server
```
